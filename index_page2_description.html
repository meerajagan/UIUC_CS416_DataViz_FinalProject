An Overview of the Data

Go to Previous Page Go to Next Page
  • The data was sourced from The Movie DataBase (TMDB.org)
  • The data was then pulled with an api call to this kaggle link: https://www.kaggle.com/datasets/asaniczka/tmdb-movies-dataset-2023-930k-movies
  • Since the database is extremely large, a sample of it was used. Movies cut off at October 2023. To determine this the following criteria were used:
  • -- English Movies were considered
  • -- Only those movies with a rating and at least 1 review were considered
  • -- Movies with duration longer than 30 mins and less than 7 hours were considered. Somes movies could be represented as a part 1 & 2 in one so that duration was considered.
  • -- Only movies with the "released" status were considerd.
  • The Alteryx flow used to clean the data can be found in the Github repo
